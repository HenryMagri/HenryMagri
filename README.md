## 📍 Projetos de Destaque

### Aplicativo de Análise de Marmoreio de Carnes

Aplicativo mobile desenvolvido para uma startup do setor agroalimentar, utilizado para análise de marmoreio em cortes de carne a partir de imagens capturadas pelo dispositivo móvel.

O aplicativo permite capturar fotos diretamente pela câmera do celular, enviar as imagens para uma API responsável pelo processamento e classificação, além de exibir histórico de análises e dashboards com visualização de resultados.

**Tecnologias**
- React Native
- Expo
- JavaScript / TypeScript
- Integração com API REST
- Captura de imagens via câmera do dispositivo
- Visualização de dados e gráficos

**Funcionalidades**
- Captura de imagens utilizando a câmera do dispositivo
- Envio de imagens para processamento em API externa
- Exibição de histórico de análises realizadas
- Dashboard com visualização gráfica dos resultados

*Projeto desenvolvido em ambiente corporativo com código proprietário.*

### Sistema Inteligente de Empréstimo de Bicicletas Elétricas

Plataforma automatizada para gerenciamento de empréstimo de bicicletas elétricas no campus universitário, utilizando **reconhecimento facial** para autenticação e validação de usuários.

O sistema foi desenvolvido de forma **end-to-end**, incluindo coleta de dados, treinamento de modelos de visão computacional, desenvolvimento de APIs e criação de um bot de WhatsApp para interação com os usuários.

**Tecnologias**
- Python
- FastAPI
- TensorFlow
- Keras
- Processamento de Imagens / Visão Computacional
- Twilio (WhatsApp Bot)
- API REST
- Integração com hardware (docas de bicicletas)

**Arquitetura**
- API de Inteligência Artificial responsável pelo processamento de imagens e reconhecimento facial
- API de automação responsável pela comunicação com o bot de WhatsApp
- Integração entre backend, sistema de autenticação e docas de bicicletas elétricas

**Responsabilidades**
- Coleta e rotulação de imagens para criação do dataset
- Treinamento de modelos de **segmentação e classificação**
- Implementação da API de IA para reconhecimento facial
- Desenvolvimento do bot de WhatsApp utilizando Twilio
- Implementação de fluxos de cadastro, autenticação e empréstimo de bicicletas
- Integração com sistema físico de liberação das bicicletas
- Implementação de mensageria por e-mail para segurança (ex: redefinição de senha)

**Funcionalidades**
- Cadastro de usuários via WhatsApp
- Coleta de imagens para treinamento do reconhecimento facial
- Validação de identidade para liberação das bicicletas
- Solicitação e devolução de empréstimos
- Redefinição de dados e gestão de conta
- Envio de notificações e recuperação de acesso por e-mail

*Projeto desenvolvido em ambiente acadêmico com código proprietário.*
